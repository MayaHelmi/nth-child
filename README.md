# CSS nth-child Exercises

## Description

A CSS practice task focused on using the `:nth-child()` selector to style specific elements without JavaScript.

## Exercises

- EX1: Zig-zag Boxes
  - Used `:nth-child(odd)` and `:nth-child(even)` to create alternating box colors and positions.

- EX2: Hide Even Numbers
  - Used `:nth-child(even)` to hide even-numbered boxes.

- EX3: Alternating Groups of Three
  - Used `:nth-child()` with formulas to create repeating groups of colored boxes.

- EX4: Select Elements
  - Practiced selecting specific elements using:
    - `:nth-child()`
    - `:nth-child(n+...)`
    - `:nth-last-child()`

## Technologies Used

- HTML5
- CSS3

## Concepts Practiced

- CSS selectors
- `:nth-child()` selector
- Flexbox
- CSS Grid
- Element positioning
- Styling without JavaScript
